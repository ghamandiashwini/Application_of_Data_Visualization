# Application of Data Visualization

## Overview

The **Application of Data Visualization** project demonstrates how data visualization techniques can be used to explore, analyze, and communicate insights from datasets. Using Python and popular data visualization libraries, the project transforms raw data into meaningful charts and graphs that support data-driven decision-making.

The project highlights the importance of visual storytelling by presenting trends, patterns, distributions, and relationships within the data.

---

## Objectives

- Clean and prepare data for visualization
- Perform exploratory data analysis (EDA)
- Create informative and visually appealing charts
- Identify trends, patterns, and outliers
- Communicate analytical findings through data visualization

## How to Run

1. Clone the repository

```bash
git clone https://github.com/ghamandiashwini/Application_of_Data_Visualization.git
```

2. Navigate to the project directory

```bash
cd Application_of_Data_Visualization
```

3. Install the required libraries

```bash
pip install pandas numpy matplotlib seaborn
```

4. Launch Jupyter Notebook

```bash
jupyter notebook
```

5. Open the notebook and run all cells.
